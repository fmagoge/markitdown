# Windows

MarkItDown requires Python 3.10 or higher.

Make a virtual environment.
```bash
python -m venv .venv
.venv\Scripts\activate
```

You can also create a virtual environment with `uv` or `Anaconda`.

To install MarkItDown, use pip: `pip install markitdown` first then install all optional depedencies `pip install markitdown[all]`
